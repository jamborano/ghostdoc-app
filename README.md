# 🛸 GhostDoc Frontend

The premium, ultra-fast frontend built with **Next.js** and **Tailwind CSS** for the GhostDoc ecosystem. Designed for hyper-performance, seamless UX, and secure client-facing operations.

## 🚀 Live Demo
Experience the live application here: **[ghostdoc.dev](https://ghostdoc.dev)**

## 🛠️ Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Deployment:** Vercel (Edge Network)
* **Architecture:** Clean Frontend Layer (Decoupled from Core Python AI Engine)
