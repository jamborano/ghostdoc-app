## The Next.js Documentation Problem

Next.js projects grow fast. With App Router, Server Components, and API routes, keeping READMEs up to date becomes impossible. Outdated docs = confused developers + slow onboarding.

**GhostDoc automates README generation for Next.js apps.**

## What GhostDoc Generates

- **Architecture maps** — Folder structure, data flow, and component relationships.
- **Setup guides** — Step-by-step installation, environment variables, and deployment.
- **Dependency graphs** — Clear visualization of package relationships.
- **API references** — Every route, middleware, and server action documented.

## How It Works

1. **Clone or ZIP upload** — Public GitHub or private ZIP (Enterprise).
2. **AI analyzes structure** — GhostDoc maps your Next.js architecture.
3. **Markdown generation** — Production-ready README.md and API_Reference.md.
4. **Encrypted delivery** — Your docs land in your inbox within 4 minutes.
